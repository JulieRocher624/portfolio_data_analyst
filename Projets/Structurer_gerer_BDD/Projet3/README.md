# Projet 3 – Requêtez une base de données avec SQL

## Bloc de compétences
RNCP37837BC01 – Structurer et gérer la base de données

## Présentation
Ce projet consiste à interroger une base de données relationnelle à l’aide de requêtes SQL afin d’extraire, filtrer et analyser des informations utiles.

## Objectifs
- Écrire des requêtes SQL
- Extraire et organiser les données
- Répondre à des besoins d’analyse simples
- Comprendre la structure d’une base relationnelle

## Outils
- SQL
- Base de données relationnelle
- PowerPoint

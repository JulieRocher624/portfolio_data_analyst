# Projet 5 – Créez et utilisez une base de données immobilière avec SQL

## Bloc de compétences
RNCP37837BC01 – Structurer et gérer la base de données

## Présentation
Ce projet consiste à concevoir, structurer et exploiter une base de données immobilière afin de répondre à des besoins d’analyse à l’aide de requêtes SQL.

## Objectifs
- Concevoir une base de données relationnelle
- Structurer les tables et les relations
- Interroger les données avec SQL
- Produire des analyses à partir d’une base immobilière

## Outils
- SQL
- Base de données relationnelle
- PowerPoint

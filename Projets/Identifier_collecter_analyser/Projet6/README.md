# Projet 6 – Optimisez la gestion des données d’une boutique avec R ou Python

## Bloc de compétences
RNCP37837BC02 – Identifier, collecter et analyser les données

## Présentation
Ce projet consiste à analyser les données d’une boutique afin d’optimiser leur gestion, d’identifier les anomalies et de formuler des recommandations d’amélioration.

## Objectifs
- Préparer et nettoyer les données
- Détecter les incohérences et anomalies
- Réaliser des analyses pertinentes
- Formuler des recommandations pour optimiser la gestion des données

## Outils
- Python ou R
- Notebook
- Pandas
- Plotly
- PowerPoint

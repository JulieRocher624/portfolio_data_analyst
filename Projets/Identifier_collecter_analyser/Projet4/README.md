# Projet 4 – Réalisez une étude de santé publique avec R ou Python

## Bloc de compétences
RNCP37837BC02 – Identifier, collecter et analyser les données

## Présentation
Ce projet consiste à analyser des données de santé publique afin d’identifier des indicateurs clés et de produire une lecture claire de la situation étudiée.

## Objectifs
- Collecter et préparer les données
- Réaliser une analyse exploratoire
- Identifier des indicateurs pertinents
- Présenter les résultats de façon claire

## Outils
- Python ou R
- Notebook
- CSV
- PowerPoint

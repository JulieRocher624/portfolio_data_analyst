# Projet 11 – Produisez une étude de marché avec R ou Python

## Bloc de compétences
RNCP37837BC05 – Réaliser des tests statistiques et concevoir des modèles d’apprentissage supervisé et non supervisé

## Présentation
Ce projet consiste à réaliser une étude de marché à partir de données, afin d’identifier des profils similaires, segmenter l’information et produire des recommandations utiles à la décision.

## Objectifs
- Préparer et explorer les données
- Réaliser une analyse statistique
- Identifier des groupes ou segments
- Produire des recommandations à partir des résultats

## Outils
- Python ou R
- Notebook
- ACP
- K-Means
- Canva

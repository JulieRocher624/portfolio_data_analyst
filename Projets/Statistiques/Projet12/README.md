# Projet 12 – Détectez des faux billets avec R ou Python

## Bloc de compétences
RNCP37837BC05 – Réaliser des tests statistiques et concevoir des modèles d’apprentissage supervisé et non supervisé

## Présentation
Ce projet consiste à analyser des caractéristiques de billets afin d’identifier les faux billets à l’aide de méthodes statistiques et de modélisation.

## Objectifs
- Préparer et explorer les données
- Identifier les variables discriminantes
- Construire un modèle d’analyse ou de prédiction
- Interpréter les résultats obtenus

## Outils
- Python ou R
- Notebook
- Script de prédiction
- Canva

# Projet 9 – Analysez les ventes d’une librairie avec R ou Python

## Bloc de compétences
RNCP37837BC05 – Réaliser des tests statistiques et concevoir des modèles d’apprentissage supervisé et non supervisé

## Présentation
Ce projet consiste à analyser les ventes d’une librairie afin d’identifier des tendances, étudier le comportement des clients et interpréter les résultats à l’aide d’analyses statistiques.

## Objectifs
- Préparer et explorer les données de ventes
- Réaliser une analyse statistique
- Identifier des tendances et relations entre variables
- Présenter les résultats de manière claire

## Outils
- Python ou R
- Notebook
- Canva

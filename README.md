# Julie Rocher – Portfolio Data Analyst

## Présentation
Bienvenue sur mon portfolio professionnel réalisé dans le cadre de ma formation **Data Analyst**.

J’ai préparé le diplôme de **Data Analyst avec OpenClassrooms** sur une durée de **deux ans**, à travers un parcours structuré autour de projets professionnalisants.  
Cette formation m’a permis de développer des compétences en **analyse de données**, **visualisation**, **reporting**, **statistiques**, **gestion de projet** et **présentation de résultats**.

## Mon parcours
En reconversion professionnelle après une première expérience en **optique**, j’ai choisi de m’orienter vers le métier de **Data Analyst** afin de mettre à profit mon goût pour la rigueur, l’analyse et la compréhension des besoins.

Durant cette formation, j’ai réalisé **13 projets professionnalisants**, organisés selon les blocs de compétences RNCP, avec des outils tels que :
- **Excel**
- **SQL**
- **SQLiteStudio**
- **Power BI**
- **Python**
- **R**
- **KNIME**
- **GitHub Pages**
- **Loom**

## Ce que vous trouverez dans ce portfolio
Ce portfolio regroupe :
- mes **projets de formation OpenClassrooms** ;
- les **livrables du projet final Aéroworld** ;
- des **tableaux de bord Power BI** ;
- des **documents de cadrage et de pilotage** ;
- une **vidéo de démonstration** ;
- une présentation synthétique de mon parcours et de mes compétences.

## Objectif
L’objectif de ce portfolio est de présenter de manière claire mon parcours, mes réalisations et les compétences développées pendant la formation, dans une logique de valorisation professionnelle.

# Dossier Aéroworld – Livrables du projet portfolio

Ce dossier regroupe les principaux livrables réalisés dans le cadre du **projet 13 – Créez votre portfolio de professionnel de la data**, autour du client fictif **Aéroworld**.

## Livrables présents

- Analyse des besoins métiers du client fictif.

- Cadrage du projet, objectifs, spécifications et contraintes.

- Planification du projet et organisation des étapes de réalisation.

- Dashboard de présentation de mon profil Data Analyst.

- Dashboard de veille sur les compétences attendues pour le métier de Data Analyst.

- Guide de création de graphiques avec Power BI.

- [Lien vers la vidéo Loom de démonstration.] (https://www.loom.com/share/b79d3c0b76a44120a883dd4c78d1f488)

## Objectif
Ces documents permettent de présenter le travail réalisé dans le cadre du portfolio, ainsi que les compétences mobilisées en analyse métier, gestion de projet, visualisation de données et communication.
